Main Repositories Under Development

Audio Signal Processing & DSP

  • JOSModules - C++ modules in JUCE Module Format to supplement standard JUCE modules
  • josm_faust - C++ Modules for JUCE Framework based on jos functions from Faust Libraries
  • faust-jos - Faust software initiated and maintained by JOS
  • modalmodel - Experiments in modal modeling
  • opengl_spectrogram - Using JUCE to create a 3D spectrogram drawn with OpenGL
  • descript-audio-codec - State-of-the-art audio codec with 90x compression factor

Machine Learning for Audio

  • WaveNet - WaveNet implementation in PyTorch with Classifier Mode
  • WaveNetAntecessor - Implementation of Wavenet in torch
  • pytorch-wavenet - An implementation of WaveNet with fast generation
  • makemore - Autoregressive character-level language model with Mamba implementation
  • AttributeModelling - Code for ML4MD paper on latent space regularization for musical attributes
  • dsp2seq - Sequence models as signal processing

Music Technology Tools

  • ace - Automatic Chord Estimation
  • MidiPredict - Play a MIDI file in advance of live incoming MIDI performance
  • 320c-starter-code - Music 320c starter code
  • Sporth - A small stack-based audio language

Development Tools

  • JUCEModulesDoc - Doxygen-format documentation for JUCE modules
  • foleys_gui_magic - A GUI builder module for JUCE with examples
  • PluginGuiMagic - Feature fork for XYDragComponent
  • juce-toys - Debugging utilities in a JUCE Module with NatVis and LLDB customizations
  • JUCEPluginTemplate - Template for JUCE plugins

Scientific Computing

  • scipy - Fork of SciPy library main repository
